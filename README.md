# How to build
Make sure you have installed [gef-cli](https://github.com/gembera/gef-cli). 

* Debug version 
```shell
yarn debug make
yarn debug build
yarn debug test
```
* Release version
```shell
yarn release make
yarn release build
yarn release test
```
